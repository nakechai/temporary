回调函数

跳到：
  1. 学到更多

回调函数是一个作为参数传递给另一个函数的函数，然后在外部函数内调用该函数来完成某种例程或动作。

这里是一个简单的例子：

function greeting(name) {
  alert('Hello ' + name);
}

function processUserInput(callback) {
  var name = prompt('Please enter your name.');
  callback(name);
}

processUserInput(greeting);

上面的例子是一个同步回调，因为它立即执行。

但是请注意，在异步操作完成后，通常会使用回调来继续执行代码- 这些称为异步回调。例如，我们简单的maps-example.html示例（也可以参考）使用Google Maps API和Geolocation API来显示设备当前位置的地图。

因为从GPS获取设备的坐标是异步的（我们不清楚数据何时会被返回），所以这个Geolocation.getCurrentPosition()方法需要一个匿名的回调函数作为参数，它本身以返回的坐标数据为参数。该函数仅在返回坐标数据时执行。
